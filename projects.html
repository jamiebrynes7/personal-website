Analysis of Activity Data from Smartphones

This project is my fourth-year design project at University of Oxford. The goal of the project is to create a software module that will harness the signals (acceleromter, gyroscope, etc) coming from a generic smartphone (Android,iOS) to provide high-accuracy physical activity markers. The algorithms that will operate on the signal data must be: robust enough to deal with different possible sets of signals, efficient enough to run on a mobile device without significant power consumption or runtime, and powerful enough to give accurate activity markers.

Currently, an Android app has been written to collect sensor data from the smartphone and email this data back to myself. In my Gmail inbox, there is a script running that will scrape the data from the email, formats it to a .csv file and deposits this file in my Google Drive.

This space will be updated as further progress is made.
