Optical NanofabricationOctober 2015 - May 2016

This project is my group third-year design project. The goal of this project is to design a fully capable fabrication facility, capable of fabricating on a sub-micron scale utilizing only optical based techniques. The facility must be able to both fabricate and image a variety of materials. I designed the optical system layout for a stereolithography process and worked to integrate this with other techniques' layouts. I also wrote the software module for the project, the core of which was tool path generation. The software took 3D model files (.stl), sliced these into layers, and generated tool paths for each layer. The functionality was wrapped up behind a Matlab GUI that could display each layer and the tool paths to the user. C++, Matlab
